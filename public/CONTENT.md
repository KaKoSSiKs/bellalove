# Контент Love Times

Файлы из `public/` доступны по путям от корня: `/music/1.mp3`, `/photos/cover/cover.jpg`.

---

## Музыка — `public/music/`

Треки подключаются в коде как **16 файлов** (по одному на секцию):

- Имена файлов: **1.mp3**, **2.mp3**, … **16.mp3**
- После добавления файлов в `src/data/audioSrc.ts` замени BASE на: `'/music/'` (локальные файлы).

Форматы: MP3 или OGG (Howler.js поддерживает оба).

---

## Фото — `public/photos/`

| Папка      | Назначение |
|-----------|------------|
| **cover/**   | Обложка (одно фото) |
| **meeting/** | Место встречи (одно фото) |
| **reasons/** | Фото для блоков «35 причин» и акцентных разворотов (можно 35+ файлов по номерам или по смыслу) |
| **eyes/**    | Слайды «Ты моими глазами» (5 фото) |
| **moments/** | Коллаж «Моменты» (6 фото) |
| **habits/**  | Привычки (6 фото) |
| **final/**   | Финальная страница (одно фото) |

В коде пути будут вида: `/photos/cover/cover.jpg`, `/photos/eyes/01.jpg` и т.д. После наполнения нужно будет обновить константы и массивы в компонентах и в `src/data/reasons.ts`.
